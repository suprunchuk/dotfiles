# Dotfiles

Personal configuration files and system settings for a productive development environment.

## Overview

This repository contains dotfiles — configuration files used to customize and set up a development environment across machines. Whether you're setting up a new system or syncing settings, these configs help maintain a consistent workflow.

## Contents

- Git configuration (`.gitconfig`)
- Terminal
- Aliases and utility scripts
